## Style Groups

- **Industrial (01-04)**: Pentagram, Stamen, Information Architects, Fathom
  - Grid-heavy, data-driven, structured
  - Best for: pricing, docs, models, technical products

- **Minimal (09-12)**: Experimental Jetset, Müller-Brockmann, Build, Sagmeister & Walsh
  - Whitespace, typography-focused, clean
  - Best for: team, about, blog, landing

- **Experimental (13-16)**: Zach Lieberman, Raven Kwok, Ash Thorp, Territory Studio
  - Motion, code-art, cinematic, futuristic
  - Best for: product hero, feature launch, creative products

## Usage

When user doesn't know the exact visual direction:
1. Show 3 options from different style groups
2. Let user pick preferred style
3. Apply picked style's prompt DNA from `references/design-styles.md`

## File Naming

`showcase-[scenario]-[style-group].html`

Example: `showcase-p-hero-industrial.html` = Product Hero + Industrial style
