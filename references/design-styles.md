## I. Information Architecture School (01-04)
> Philosophy: "Data is not decoration, it's building material"

### 01. Pentagram - Michael Bierut Style
**Philosophy**: Type is language, grid is thought
**Key Traits**:
- Extremely restrained colors (black/white + 1 brand color)
- Modern Swiss grid system interpretation
- Typography as primary visual language
- Strategic use of negative space (60%+ whitespace)

**Prompt DNA**:
```
Pentagram/Michael Bierut style:
- Extreme typographic hierarchy, Helvetica/Univers family
- Swiss grid with precise mathematical spacing
- Black/white + one accent color (#HEX)
- Information architecture as visual structure
- 60%+ whitespace ratio
- Data visualization as primary decoration
```

**Representative**: Hillary Clinton 2016 campaign identity
**Search Keywords**: pentagram hillary logo system

## Prompt Usage Guide

**Combination Formula**: `[Style Prompt DNA] + [Scene Template] + [Specific Content]`

### Core Principle: Describe Mood, Not Layout

Key to AI image generation: short prompt > long prompt. Describing mood and content in 3 sentences works better than 30 lines of layout details.

| Kill Diversity | Boost Creativity |
|---------------|-----------------|
| Specify color ratios (60%/25%/15%) | Describe mood ("warm like Sunday morning") |
| Specify layout position ("title centered, image right") | Reference specific aesthetics ("Pentagram editorial feel") |
| Limit character poses and expressions | Let AI naturally interpret style |
| List all visual elements to include | Describe what audience should FEEL |

### Good / Bad Examples

**Bad — Over-constrained (output flat and empty)**:
```
Professional presentation slide. Dark background, light text.
Title centered at top. Two columns below. Left column: bullet points.
Right column: bar chart. Colors: navy 60%, white 30%, gold 10%.
Font size: title 36pt, body 18pt. Margins: 40px all sides.
```

**Good — Mood-driven (diverse and textured)**:
```
A data visualization that feels like a Bloomberg Businessweek
editorial spread. The key number "28.5%" should dominate the
composition like a headline. Warm cream tones with sharp black
typography. The data tells a story of dramatic channel shift.
```

### Execution Path Selection

Choose based on "Best Path" column in quick reference:
- **AI Generation**: Clear visual styles (06/07/12/13/14/15/16/20), use Gemini/Midjourney
- **HTML Rendering**: Precise typography styles (01/03/04/10/11/17/18), code controls layout
- **Hybrid**: HTML skeleton + AI backgrounds/images (02/05/08/09/19)

### Quality Control

1. ❌ Don't write "in the style of Pentagram" → ✅ Describe specific design traits
2. Text often errors in AI generation → Replace text after generation
3. Ratios easily distorted → Specify aspect ratio explicitly
4. Generate 3-5 variants first, select best, then refine

**Default Aesthetic No-Go Zone** (user can override with their brand):
- ❌ Cyber neon/dark blue background (#0D1117)
- ❌ Add personal signature/watermark to cover images
